## Installation

clone o repositório 
```sh
https://github.com/Reginaldo1239/gerenciar_funcionarios.git
```

rode o seguinte comando para instalar as dependencias
```sh
npm install;
```
para iniciar o servidor fake
```sh
json-server --watch db.json --port 3004
```
iniciar o projeto 
```sh
npm start;
```

rodar os testes unitários 
```sh
npm test;
```
rodar os testes e2e
```sh
npx cypress open
```
